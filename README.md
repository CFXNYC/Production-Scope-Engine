# Project Scope Intelligence Engine

This is a React + TypeScript + Tailwind/Vite prototype.

## Launch

Double-clicking `index.html` will not launch the app correctly because the React code needs to be served by Vite.

Use one of these:

```bash
npm run dev -- --host 127.0.0.1
```

Then open:

```txt
http://127.0.0.1:5173/
```

Or double-click:

```txt
LAUNCH_APP.command
```
